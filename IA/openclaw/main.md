# OpenClaw Knowledge Base

OpenClaw is an open-source personal AI assistant that runs on your own devices. It connects AI models with your local files, messaging apps, and system capabilities.

## Quick Start

```bash
# Install
curl -fsSL https://openclaw.ai/install.sh | bash

# Onboard
openclaw onboard --install-daemon

# Check status
openclaw gateway status
```
