# I/O and Files

Related: [Environment](13-environment), [Pipelines](11-pipelines).

## File Commands

```nu
# Read file
open file.txt

# Read lines
open file.txt | lines

# Write file (overwrite)
"content" | save file.txt

# Append
"content" | save --append file.txt

# Binary mode
open file.bin --raw
```

## Standard I/O

```nu
# Pipeline to external
echo "hello" | xargs echo

# Capture output
$(echo hello)  # as string

# Run external
^ls -la

# Background
^Background
```

## HTTP

```nu
# GET request
http get "https://api.example.com/data"

# POST request
http post "https://api.example.com/data" {field: "value"}

# With headers
http get "https://api.example.com" -H [Authorization "Bearer token"]
```

## CSV and JSON

```nu
# Read CSV
open data.csv | from csv

# Write CSV
$data | to csv

# Read JSON
open data.json | from json

# Pretty print
$data | to json
```

## HTML/XML

```nu
# From HTML table
curl "url" | from html --fragments

# To TOML
$data | into record | to toml
```
