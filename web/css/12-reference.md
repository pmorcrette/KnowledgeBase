# CSS Reference

Common CSS properties quick reference.

## Quick Properties

```css
/* Display */
display: none | block | inline | inline-block | flex | grid | contents;

/* Overflow */
overflow: visible | hidden | scroll | auto;

/* Float */
float: left | right | none;

/* Clear */
clear: left | right | both;

/* Cursor */
cursor: pointer | default | grab | not-allowed;
```
